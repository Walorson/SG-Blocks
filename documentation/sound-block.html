Sound Block

  • The Block that plays a sound given by user.

Properites

  • Category (Default: other): The category of sounds.
  • Sound: List of sound that you can choose.
  • Wait: How long block will wait before the next step
    • Run Speed time: Default amount of time given by user in Start Block.
    • Until the sound ends: When the sound ends the block moves to the next step.
    • Custom: Custom time provided by user.
  • Wait time (ms): If you choice "Custom" wait mode, you can enter the amount of time the block will wait before the next step.
