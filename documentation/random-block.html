Random Block

  • The Block that draw a random number from the range of numbers provided by the user.

Properites

  • Min (Default: 1) - lower bound of the range that block randomizes
  • Max (Default: 100) - upper bound of the range that block randomizes
  • Save to Variable (Default: var1) - a variable in which a random number is saved

Use

  • Randomizing an integer number within the given range
  • Creating simple game mechanics (e.g dice rolling, lottery)
  • Statistical simulations

Random: 1 - 100
