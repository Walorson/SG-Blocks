Input Block

  • The Block that enables to enter a variable from the keyboard by the user.

Properites

  • Variable Name (Default: var1): Variable where the input is saved.
  • Message (Default: Enter variable): Message that appears while you enter an input
  • Input Mode (Default: textarea): Mode of the input.
    • Textarea- input data is entered from the keyboard
    • Boolean- input data is entered by pressing buttons "Yes" or "No".

Use

  • Entering variable from keyboard and their subsequent processing
  • User decision-making
