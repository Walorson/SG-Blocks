Condition Block

  • The Block that enables to check a condition and make decision by true/false line.

Properites

  • Number field: A field to type a number / text.
  • Operator type: A select list to choose an operator type. (e.g. ==, >)
  • Var checkbox: A checkbox that you decide if you want to use a variable for comparison or a number / text that you type from keyboard.
  • Add a condition: A button that creates you a next condition (You've got a multiple condition)

Use

  • To make various calculations on numerical variables.
